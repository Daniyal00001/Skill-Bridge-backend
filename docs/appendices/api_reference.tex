\chapter{API Endpoint Reference}
\label{app:api}

This appendix documents all REST API endpoints exposed by the \projectname{} backend. Protected routes require a \texttt{Bearer} JWT access token in the \texttt{Authorization} header unless stated otherwise.

\section*{Legend}

\begin{itemize}
  \item \textbf{[PUB]} â€” Public (no authentication required)
  \item \textbf{[AUTH]} â€” Requires authentication (any role)
  \item \textbf{[CLIENT]} â€” Requires CLIENT role
  \item \textbf{[FRLNC]} â€” Requires FREELANCER role
  \item \textbf{[ADMIN]} â€” Requires ADMIN role
\end{itemize}
